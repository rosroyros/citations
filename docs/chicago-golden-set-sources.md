# Chicago Golden Set Sources

## Overview

This document identifies verified university library guides for extracting Chicago 17th Edition Notes-Bibliography citation examples. These sources are "non-contaminated" - meaning they are not primary citation reference sites that LLMs were heavily trained on (like Purdue OWL, Scribbr, EasyBib).

**Verification Date**: 2025-12-31
**Target**: 8+ universities with real examples
**Result**: 8 universities verified with 250+ combined examples
