# AUDIT WORKSHEET: 94 Remaining Citations
## Computer-Assisted Manual Review

**INSTRUCTIONS:**
This worksheet contains an automated preliminary audit. However, automated APA checking has limitations:
- Cannot distinguish journal names (title case) from article titles (sentence case)
- Cannot see italic formatting in plain text
- Cannot judge context-specific rules

**For each citation:**
1. Review the automated findings (if any)
2. Check against APA 7 rules yourself using https://libguides.css.edu/APA7thEd/
3. Mark your final decision

**APA 7 Quick Reference:**
- **Sentence case:** Only first word + first word after colon + proper nouns capitalized
- **Book titles:** Sentence case, italicized
- **Article titles:** Sentence case, NOT italicized
- **Journal names:** Title case (major words capitalized), italicized
- **Volume:** Italicized
- **Issue:** Parentheses, NOT italicized
- **DOI:** Must include https://doi.org/ (http://dx.doi.org/ is old format)
- **Author initials:** Space after period (A. B. Smith, not A.B.Smith)
- **Date format:** (Year, Month Day) with comma
- **En-dash vs hyphen:** IGNORE (treat as equivalent per task notes)
